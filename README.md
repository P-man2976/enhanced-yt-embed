## これは何？

YouTubeの埋め込みプレーヤーで、YouTube上の動画を連続再生・シャッフル・リピートできます

（JotaiとTailwind CSSを使う練習をするために作成しました）

## 構成

- React
- Tailwind CSS
- Jotai
- Tanstack Query
